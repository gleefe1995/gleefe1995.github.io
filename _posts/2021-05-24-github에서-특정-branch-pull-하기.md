branch 사용하는 방법



### branch 확인

```
git branch
```



### branch 만들기

```
git branch branchname
```

### branch 전환

```
git checkout branchname
```

### 원격에서 branch pull 해오기

```
sudo git pull
git checkout -t origin/branchname
```
